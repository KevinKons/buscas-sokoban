Antes de iniciar o software, tenha certeza de, na mesma pasta do executável, conter as instâncias disponibilizadas pelos alunos responsáveis pelo projeto.

Para executar o software, você irá seguir os seguintes passos:

1 - Abrir o terminal e navegar até a pasta onde encontra-se o executável;

2 - Digitar no terminal: java -jar sokoban.jar;

3 - Escolha uma das opções solicitadas no terminal;

4 - Esperar a finalização do algoritmo. Você será notificado no terminal quando isso ocorrer;

5 - Na mesma pasta que está o executável, será gerado um novo arquivo chamado output.exe com as informações sobre cada instância carregada, e para cada instância, informações sobre cada algoritmo, como o tempo de execução até encontrar a solução, o número de nós visitados e o nome da busca que contem essas informações;
